아래는 내가 개발한 AI 기반 영유아 육아 기록 앱 프로젝트의 전체 내용입니다.
이를 바탕으로 발표용 PPT 슬라이드 구성안을 작성해주세요.

## 프로젝트 개요
프로젝트명: 아기의 기록 (Baby Record App)
개발 목적: 영유아 부모의 일상적인 육아 기록 부담을 줄이고, 아기 성장 단계에 맞는 맞춤 정보를 제공하는 AI 기반 육아 플랫폼 개발
기술 스택: Next.js, React, TypeScript, MySQL (Supabase/Vercel 배포), OpenAI GPT-4o-mini, YouTube Data API, OpenWeatherMap API, Web Speech API

■ 프로젝트 한 줄 요약
영·유아(0~24개월) 부모를 위한 육아 올인원 모바일 웹앱 —
일상 기록 · 발달 체크 · AI 챗봇 · 알람이 하나의 앱에 담긴 서비스

■ 핵심 사용자 문제 (기획 단계 정의)
1. 기록의 번거로움 — 수면·수유·배변을 매번 따로 기록
2. 시기 놓침 — 예방접종, 이유식 시작 등 타이밍을 모름
3. 정보 탐색 피로 — 발달 정보를 매번 인터넷 검색
4. 진료 연계 불편 — 비대면 진료 시 아이 데이터 정리 번거로움
---
■ 실제 구현된 주요 기능 (기획 초과 달성 항목 포함)

[기록]
- 음성 인식 + GPT-4o-mini 인텐트 분류로 자연어 기록
  예) "아기 방금 분유 150ml 먹었어" → 수유 기록 자동 저장
- 수유 시작/완료 실시간 추적 (진행 중 트래킹)
- 수면 시작/종료 자동 계산 + 소요 시간 표시
- 시작 시간 ±1/10/30분 빠른 조정 버튼 (육아 중 한 손 조작 UX)
- 완료 시간 직접 입력 vs 소요 시간 버튼 중 선택 가능

[스케줄 & 달력]
- 아기 개월수 기준 예방접종·영유아검진·이유식·발달 마일스톤 자동 시드
  (날짜 미지정 — 부모가 직접 예약일 설정, 혼란 방지)
- 홈 탭 다가오는 일정: 오늘 기준 가장 가까운 일정 3개 + 날짜 뱃지
- 스케줄 탭 ↔ 홈 탭 ↔ 육아달력 3중 연동
- 일정 클릭 → 수정 모달 (날짜·카테고리·내용 변경, 삭제)

[AI 챗봇 — RAG 기반]
- 8개 국내 육아 전문 가이드라인 문서 벡터 검색 (RAG)
- 아기 이름·개월수 컨텍스트 자동 포함 맞춤 응답
- 심폐소생술 가이드라인: 명시적 질의 시에만 참조 (안전 필터)
- 오프라인 fallback 응답 함수 내장

[건강 & 리포트]
- WHO P3/P50/P97 성장 곡선 SVG 시각화 (성별 구분)
- 예방접종 이력 관리 표: 아기수첩 관리 불편함 해소
  (BCG·B형간염·DTaP·폴리오·Hib·폐구균·수두·MMR + 로타·독감·HPV 등)
- 건강 리포트 PDF: 생활패턴·통계·성장곡선·건강이슈·복약·예방접종 3페이지
- 공유: 모바일 네이티브 공유 시트(카카오톡 포함) / PC 공유 오버레이

[정보 탭]
- 아기 개월수 기반 YouTube 검색 쿼리 자동 생성 (먹기·자기·놀기·건강)
- 카테고리 내 키워드 추가 / 사용자 커스텀 카테고리 추가
- 맞춤 키워드: 현재 개월 + 다음 1개월 키워드 합산 제공

[발달]
- 개월별 발달 마일스톤 체크리스트
- 날짜별 발달 메모 (음성 입력·수정·삭제)


## 핵심 기능 (10가지)

1. 음성 기반 시간표 기록
   - Web Speech API + GPT-4o-mini 인텐트 분류로 "아기 방금 분유 150ml 먹었어" 같은 자연어를 시간표에 자동 기록
   - 수유, 수면, 기저귀, 울음, 산책, 놀이, 목욕, 키/몸무게 모두 음성으로 기록
   - 수유 시작/완료 추적, 수면 시작/완료 실시간 추적

2. 아기 개월수 기반 맞춤 일정 자동 시드
   - 아기 생년월일 입력 시 현재 개월수 기준 ±3개월 이내 예방접종, 영유아검진, 이유식 시작, 발달 마일스톤 일정 자동 생성
   - baby_develop_db (자체 구축 DB) 기반 개월별 핵심 발달/건강 키워드 제공
   - 자동 시드 일정은 날짜 미지정 → 부모가 직접 병원 예약일 설정

3. AI 챗봇 (RAG 기반)
   - 8개 육아 전문 가이드라인 문서를 RAG(검색 증강 생성)로 학습
   - 아기 이름·개월수 컨텍스트 포함 맞춤 응답
   - 심폐소생술 가이드라인은 명시적 질의 시에만 참조 (안전 필터링)

4. 성장 곡선 & 백분위
   - WHO P3/P50/P97 기준 키·몸무게 성장 곡선 SVG 시각화
   - 최근 측정치 백분위 자동 계산 (성별 구분)

5. 발달 체크리스트 + 메모
   - 개월별 발달 마일스톤 체크리스트
   - 날짜별 발달 메모 (음성 입력 지원, 수정/삭제 가능)

6. 건강 리포트 PDF & 공유
   - 생활 패턴 차트, 평균 수유량·수면시간, 성장 곡선, 건강이슈, 복약정보, 예방접종 이력을 A4 3페이지 PDF로 출력
   - 모바일: 네이티브 공유 시트(카카오톡 등) / PC: 카카오톡·이메일·링크 복사 지원

7. 예방접종 이력 관리
   - 기본접종 8종 + 선별접종 6종 + 사용자 커스텀 접종 추가
   - 횟수별(1~5차) 접종일·병원·제품명 기록, 표 형태 시각화

8. 유튜브 개월수 맞춤 추천
   - 아기 개월수에 따라 먹기·자기·놀기·건강·발달 카테고리별 검색 쿼리 자동 생성
   - 사용자 커스텀 카테고리 추가, 카테고리 내 키워드 추가 가능

9. 스케줄 탭 ↔ 홈 탭 ↔ 육아달력 연동
   - 모든 일정은 동일한 데이터 공유
   - 홈 탭: 오늘 기준 가장 가까운 일정 3개 표시 (날짜 뱃지 색상 구분)
   - 육아달력: 예방접종·이유식·검진 날짜 자동 표시 + 사용자 일정 점(●) 표시

10. 시간 기록 UX 혁신
    - 시작 시간 ±1/10/30분 빠른 조정 버튼
    - 소요 시간 버튼 (+1/5/10/30분) 또는 완료 시간 직접 입력 선택
    - 두 방식 모두 소요 시간 자동 계산 및 표시

## 타깃 사용자
- 신생아~36개월 영유아 부모 (특히 초보 부모)
- 맞벌이 부부, 조부모 공동 육아 가정
- 발달 걱정이 있는 부모, 예방접종 일정 관리가 필요한 가정

## 차별점
- 단순 기록 앱이 아닌 AI 챗봇 + RAG + 음성인식 통합
- 아기 개월수 기반 자동 맞춤화 (일정, 키워드, 유튜브 추천 모두 연동)
- baby_develop_db 자체 구축으로 국내 의료 가이드라인 기반 정보 제공
- PDF 리포트로 소아과 방문 시 활용 가능

## 향후 발전 방향
- 카카오톡/앱 알림으로 예방접종·발달 키워드 알림
- 앱 설치자 집단 통계 기반 익명 발달 비교
- 소아과 연동 및 진료 기록 가져오기
- 형제자매 복수 아기 지원
